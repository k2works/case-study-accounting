## 概要

財務会計システムの最初のリリースとして、基本的な仕訳入力と元帳照会ができる最小限の機能を提供します。ユーザー認証、勘定科目マスタ管理、仕訳の CRUD 操作、総勘定元帳・日次残高照会を含みます。

## アップグレード手順

初回リリースのため、アップグレード手順はありません。

### インストール手順

1. リポジトリをクローン
2. Docker Compose で環境構築: `docker-compose up -d`
3. バックエンド起動: `cd apps/backend && ./gradlew bootRun`
4. フロントエンド起動: `cd apps/frontend && npm run dev`
5. ブラウザで `http://localhost:3000` にアクセス
