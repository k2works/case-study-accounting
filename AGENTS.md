# サブエージェント実行ガイドライン

このファイルは Claude Code のサブエージェント（Task ツールで起動されるエージェント）向けの指示です。

## 基本原則

**CLAUDE.md の全ての指示に従うこと。**

サブエージェントは CLAUDE.md に記載された以下の内容を継承します：

- ペルソナ
- コア原則
- 基本設定（言語、文体、スペースルール）
- 実行ルール
- 開発手法（TDD サイクル、変更管理）
- 品質保証

## サブエージェント固有の指示

### コンテキストの効率化

- 必要な情報のみを収集し、簡潔に報告する
- 冗長な説明は避け、結果を端的にまとめる
- 大量のコード出力は避け、要点を抽出する

### 報告形式

タスク完了時は以下の形式で報告：

```markdown
## 結果

[タスクの結果を簡潔に記述]

## 詳細

[必要に応じて詳細情報]
```

### 参照ドキュメント

以下のドキュメントを必要に応じて参照：

- `CLAUDE.md` - メインの実行ガイドライン
- `docs/reference/開発ガイド.md` - 開発ライフサイクル
- `docs/reference/よいソフトウェアとは.md` - 品質基準

### 禁止事項

- CLAUDE.md の指示に反する行動
- ユーザーへの直接的な質問（親エージェントを通じて行う）
- 不必要なファイルの作成
- 過度に長い出力
